# Mapa de Pontos de Poda com Clusterização e Rastreamento

Este projeto exibe pontos de poda em um mapa interativo usando **Leaflet**, **PapaParse** e **MarkerCluster**, com opção de rastrear a localização do usuário em tempo real.

## Funcionalidades
- Clusterização para melhorar desempenho com milhares de pontos.
- Rastreamento da localização do usuário com botão para ativar/desativar.

## Observações
- O rastreamento só funciona via HTTPS ou localhost.
- O arquivo CSV deve estar no mesmo repositório.
