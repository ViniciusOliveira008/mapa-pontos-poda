# Mapa de Pontos de Poda com Clusterização e Localização Automática

Este projeto exibe pontos de poda em um mapa interativo usando **Leaflet**, **PapaParse** e **MarkerCluster**, com localização automática do usuário.

## Funcionalidades
- Clusterização para melhorar desempenho com milhares de pontos.
- Localização automática do usuário (ativa quando permitido).

## Observações
- O rastreamento só funciona via HTTPS ou localhost.
- O arquivo CSV deve estar no mesmo repositório.
