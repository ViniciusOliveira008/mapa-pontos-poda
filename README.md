# Mapa de Pontos de Poda com Ícones Dinâmicos

Este projeto exibe pontos de poda em um mapa interativo usando **Leaflet**, **PapaParse** e **MarkerCluster**, com localização automática do usuário e ícones que aumentam conforme o zoom.

## Funcionalidades
- Clusterização para melhorar desempenho com milhares de pontos.
- Localização automática do usuário.
- Ícones dinâmicos que aumentam até 50px conforme o zoom.

## Observações
- O rastreamento só funciona via HTTPS ou localhost.
- O arquivo CSV deve estar no mesmo repositório.
